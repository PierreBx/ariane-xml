# Welcome to Ariane-XML

**Ariane-XML** is a powerful SQL-like query engine for XML documents, designed for data analysts and developers who need to extract and analyze data from complex XML structures.

## Key Features

- **SQL-like Syntax**: Query XML files using familiar SELECT, FROM, WHERE, ORDER BY, and LIMIT clauses
- **DSN Mode**: Specialized support for French DSN (DÃ©claration Sociale Nominative) schemas
- **Rich Output**: Beautiful HTML tables with sorting, filtering, and export capabilities
- **Encryption Module**: Format-Preserving Encryption and French data pseudonymization
